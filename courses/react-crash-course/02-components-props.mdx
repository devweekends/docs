# Components & Props

Components are the building blocks of any React application. They let you split the UI into independent, reusable pieces, and think about each piece in isolation.

## Functional Components

In modern React, we primarily use **Functional Components**. They are simply JavaScript functions that return JSX.

```javascript
function Welcome() {
  return <h1>Hello, World</h1>;
}

// Or using Arrow Functions
const Welcome = () => {
  return <h1>Hello, World</h1>;
};
```

To use this component in another file:

```javascript
import Welcome from './Welcome';

function App() {
  return (
    <div>
      <Welcome />
    </div>
  );
}
```

## Props (Properties)

Props are arguments passed into React components. They are passed to components via HTML attributes.

### Passing Props

```javascript
function App() {
  return <Welcome name="Sara" age={25} />;
}
```

### Receiving Props

The component receives the props as an object.

```javascript
function Welcome(props) {
  return <h1>Hello, {props.name}. You are {props.age} years old.</h1>;
}
```

### Destructuring Props

It is common practice to destructure props for cleaner code.

```javascript
function Welcome({ name, age }) {
  return <h1>Hello, {name}. You are {age} years old.</h1>;
}
```

## Children Prop

`props.children` is a special prop that allows you to pass components or content *inside* another component.

```javascript
function Card({ children }) {
  return <div className="card">{children}</div>;
}

function App() {
  return (
    <Card>
      <h2>Title</h2>
      <p>This is content inside the card.</p>
    </Card>
  );
}
```

## Default Props

You can define default values for props.

```javascript
function Button({ color = 'blue', text }) {
  return <button style={{ backgroundColor: color }}>{text}</button>;
}
```

## Summary

- **Components** are reusable UI pieces.
- **Functional Components** are functions that return JSX.
- **Props** allow passing data from parent to child components.
- Props are **read-only** (immutable).
- Use **destructuring** to access props easily.
- `children` prop is used for content nesting.

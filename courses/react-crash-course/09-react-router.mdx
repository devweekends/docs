# React Router

React Router is the standard routing library for React. It enables the navigation among views of various components in a React Application, allows changing the browser URL, and keeps the UI in sync with the URL.

## Installation

```bash
npm install react-router-dom
```

## Basic Setup

Wrap your app in `BrowserRouter`.

**main.jsx**
```javascript
import React from 'react';
import ReactDOM from 'react-dom/client';
import { BrowserRouter } from 'react-router-dom';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')).render(
  <BrowserRouter>
    <App />
  </BrowserRouter>
);
```

## Defining Routes

Use `Routes` and `Route` to define your page structure.

**App.jsx**
```javascript
import { Routes, Route, Link } from 'react-router-dom';
import Home from './Home';
import About from './About';
import Contact from './Contact';

function App() {
  return (
    <div>
      <nav>
        <Link to="/">Home</Link> | <Link to="/about">About</Link> | <Link to="/contact">Contact</Link>
      </nav>

      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/contact" element={<Contact />} />
      </Routes>
    </div>
  );
}
```

## Dynamic Routing

You can pass parameters in the URL.

```javascript
<Route path="/users/:id" element={<UserProfile />} />
```

### Accessing Parameters

Use the `useParams` hook.

**UserProfile.jsx**
```javascript
import { useParams } from 'react-router-dom';

function UserProfile() {
  const { id } = useParams();
  return <h1>User ID: {id}</h1>;
}
```

## Navigation

### Link Component

Always use `<Link>` instead of `<a>` tags to prevent full page reloads.

```javascript
<Link to="/about">Go to About</Link>
```

### Programmatic Navigation

Use the `useNavigate` hook to redirect users via code (e.g., after form submission).

```javascript
import { useNavigate } from 'react-router-dom';

function Login() {
  const navigate = useNavigate();

  const handleLogin = () => {
    // ... login logic
    navigate('/dashboard');
  };

  return <button onClick={handleLogin}>Login</button>;
}
```

## 404 Page (Not Found)

Add a catch-all route at the end.

```javascript
<Route path="*" element={<NotFound />} />
```

## Summary

- **React Router** enables SPA (Single Page Application) routing.
- Wrap app in `BrowserRouter`.
- Use `Routes` and `Route` to define paths.
- Use `Link` for navigation links.
- Use `useParams` to access URL parameters.
- Use `useNavigate` for programmatic redirection.

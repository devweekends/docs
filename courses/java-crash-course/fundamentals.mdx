# Java Fundamentals

Welcome to Java! This chapter covers the essential building blocks of the language. If you're new to programming, start here and experiment with the examples. You'll learn about variables, types, control flow, loops, input/output, and more.

---

## Hello World

The classic first program. Every Java application starts with a `main` method:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

**Explanation:**
- `public class HelloWorld` defines a class named HelloWorld.
- `public static void main(String[] args)` is the entry point. The code inside runs when you start the program.
- `System.out.println(...)` prints text to the console.

## Data Types & Variables


Java is statically typed. You must declare the type of each variable. There are two main categories:

- **Primitive types**: `int`, `double`, `char`, `boolean`, `byte`, `short`, `long`, `float`
- **Reference types**: objects, arrays, classes (e.g., `String`, `Person`)

```java
int age = 25;           // Integer (primitive)
double price = 19.99;   // Decimal number (primitive)
char grade = 'A';       // Single character (primitive)
boolean isActive = true; // true or false (primitive)
String name = "Alice";  // Reference type (object)
```

**Type inference** (Java 10+):
```java
var x = 42;    // int
var y = 3.14;  // double
var message = "Hello"; // String
```

**Pro tip:** Use primitive types for simple values, reference types for objects and collections.

## Control Flow

Control flow lets you make decisions and repeat actions.

### If-Else
```java
if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C or below");
}
```

### Switch Expressions (Java 14+)
```java
String dayName = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    default -> "Other";
};
```

---

## Loops

Loops let you repeat code. There are several types:

**For loop:**
```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

**While loop:**
```java
int count = 0;
while (count < 3) {
    System.out.println("Count: " + count);
    count++;
}
```

**Enhanced for loop (for-each):**
```java
int[] numbers = {1, 2, 3};
for (int n : numbers) {
    System.out.println(n);
}
```

## Input and Output

To read user input, use `Scanner`:
```java
import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
System.out.print("Enter your name: ");
String name = scanner.nextLine();
System.out.println("Hello, " + name + "!");
```

## Comments

Use comments to explain your code:
```java
// This is a single-line comment
/*
   This is a multi-line comment
*/
```

## Error Handling

Java uses exceptions to handle errors:
```java
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Error: " + e.getMessage());
}
```

---

## Best Practices

- Use meaningful variable names (`age`, `price`)
- Indent code for readability
- Comment your code to explain logic
- Prefer `final` for constants: `final double PI = 3.14;`
- Avoid magic numbers (use named constants)

---

## Common Pitfalls

- Forgetting to initialize variables
- Using `==` to compare strings (use `equals()` instead)
- Array indices start at 0

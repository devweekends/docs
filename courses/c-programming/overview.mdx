# Advanced C Programming for Systems Development

A hardcore curriculum for programmers who want to **build operating systems, compilers, databases, and embedded firmware**. We move fast through basics and dive deep into what makes C the backbone of computing infrastructure.

<Info>
**Course Duration**: 14-18 weeks (self-paced)  
**Target Outcome**: Systems Programmer / Kernel Developer / Embedded Engineer  
**Prerequisites**: Prior programming experience (any language)  
**Primary Focus**: Low-level systems programming, memory mastery, performance
</Info>

---

## Track 3: Advanced Concepts

Master the features that separate junior from senior C programmers.

<AccordionGroup>
  <Accordion title="Module 8: Preprocessor Mastery" icon="code">
    **Duration**: 4-6 hours
    
    - Macro hygiene and best practices
    - X-macros for code generation
    - Include guards and pragma once
    - Conditional compilation strategies
    - Variadic macros
    - Stringification and token pasting
  </Accordion>

  <Accordion title="Module 9: Data Structures in C" icon="sitemap">
    **Duration**: 10-12 hours
    
    - Linked lists with intrusive containers
    - Hash tables (open addressing, chaining)
    - Binary trees and red-black trees
    - Generic containers with void pointers
    - Linux kernel container_of macro
  </Accordion>

  <Accordion title="Module 10: Function Pointers & Callbacks" icon="arrow-right-arrow-left">
    **Duration**: 4-6 hours
    
    - Function pointer syntax and typedef
    - Callback patterns
    - Jump tables and dispatch tables
    - Closures with context pointers
    - Plugin architectures
  </Accordion>

  <Accordion title="Module 11: Undefined Behavior" icon="skull">
    **Duration**: 6-8 hours
    
    - Signed overflow
    - Null pointer dereference
    - Buffer overflows
    - Use after free
    - Data races
    - Strict aliasing violations
    - Unsequenced modifications
    - How compilers exploit UB for optimization
  </Accordion>
</AccordionGroup>

## Track 4: Systems Programming

Real-world systems development with POSIX APIs.

<AccordionGroup>
  <Accordion title="Module 12: System Calls & POSIX" icon="terminal">
    **Duration**: 8-10 hours
    
    - User space vs kernel space
    - System call mechanics (syscall instruction)
    - Error handling with errno
    - POSIX standards and portability
  </Accordion>

  <Accordion title="Module 13: File I/O & Filesystems" icon="folder-open">
    **Duration**: 8-10 hours
    
    - Low-level I/O: open, read, write, close
    - File descriptors and the fd table
    - Buffered vs unbuffered I/O
    - Memory-mapped I/O
    - Directory operations
    - inotify for file watching
  </Accordion>

  <Accordion title="Module 14: Process Programming" icon="microchip">
    **Duration**: 8-10 hours
    
    - fork(), exec(), wait() family
    - Process creation and termination
    - Signal handling
    - Daemon processes
    - Process groups and sessions
  </Accordion>

  <Accordion title="Module 15: Thread Programming" icon="arrows-split-up-and-left">
    **Duration**: 10-12 hours
    
    - POSIX threads (pthreads)
    - Thread creation and lifecycle
    - Mutexes and condition variables
    - Reader-writer locks
    - Thread-local storage
    - Thread pools
  </Accordion>

  <Accordion title="Module 16: Network Programming" icon="network-wired">
    **Duration**: 10-12 hours
    
    - Socket programming fundamentals
    - TCP client/server architecture
    - UDP programming
    - Non-blocking I/O
    - select/poll/epoll
    - High-performance event loops
  </Accordion>

  <Accordion title="Module 17: IPC & Shared Memory" icon="share-nodes">
    **Duration**: 6-8 hours
    
    - Pipes and FIFOs
    - POSIX shared memory
    - Message queues
    - Semaphores
    - Memory barriers and atomics
  </Accordion>
</AccordionGroup>

## Track 6: Real-World Projects

Build serious infrastructure from scratch.

<AccordionGroup>
  <Accordion title="Project 1: Memory Allocator" icon="memory">
    **Duration**: 15-20 hours

    - Free list management
    - Coalescing free blocks
    - Splitting blocks
    - Best fit vs first fit

  </Accordion>

  <Accordion title="Project 2: Unix Shell" icon="terminal">
    **Duration**: 15-20 hours

    - Command parsing
    - Process creation and management
    - Pipes and redirection

  </Accordion>

  <Accordion title="Project 3: Key-Value Database" icon="database">
    **Duration**: 20-30 hours

# Pointers & Memory

## Pointers
```cpp
int x = 42;
int* ptr = &x;
std::cout << *ptr; // 42
```

## References
```cpp
int x = 10;
int& ref = x;
ref = 20; // x is now 20
```

## Dynamic Memory
```cpp
// new/delete
int* p = new int(42);
delete p;

// Smart pointers (C++11)
std::unique_ptr<int> up = std::make_unique<int>(42);
std::shared_ptr<int> sp = std::make_shared<int>(42);
```

## RAII
```cpp
class File {
    FILE* file;
public:
    File(const char* name) { file = fopen(name, "r"); }
    ~File() { if (file) fclose(file); }
};
```

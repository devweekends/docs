# C++ Fundamentals

## Hello World
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## Data Types
```cpp
int age = 25;
double price = 19.99;
char grade = 'A';
bool isActive = true;

// Auto (C++11)
auto x = 42;
auto y = 3.14;
```

## Control Flow
```cpp
// If-else
if (score >= 90) {
    std::cout << "A";
} else if (score >= 80) {
    std::cout << "B";
}

// For loop
for (int i = 0; i < 10; i++) {
    std::cout << i << " ";
}

// Range-based for (C++11)
std::vector<int> nums = {1, 2, 3, 4, 5};
for (const auto& num : nums) {
    std::cout << num << " ";
}
```

## Functions
```cpp
int add(int a, int b) {
    return a + b;
}

// Lambda (C++11)
auto multiply = [](int a, int b) { return a * b; };
```

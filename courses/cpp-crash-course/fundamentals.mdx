# C++ Fundamentals

## Hello World
The classic starting point! In C++, you need to include headers and define a `main` function. `std::cout` is used for output.

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl; // Print to console
    return 0; // Exit code
}
```

**Tip:** Every C++ program starts from `main()`. `std::endl` flushes the output buffer and adds a newline.

---

## Data Types
C++ is statically typed. You must declare the type of every variable.

```cpp
int age = 25;           // Integer
double price = 19.99;   // Floating point
char grade = 'A';       // Single character
bool isActive = true;   // Boolean

// Auto (C++11+): Let the compiler deduce the type
auto x = 42;    // int
auto y = 3.14;  // double
```

**Common Pitfall:** Uninitialized variables have undefined values! Always initialize.

---

## Control Flow
Control the logic of your program with if/else, loops, and switches.

### If-Else
```cpp
if (score >= 90) {
    std::cout << "A";
} else if (score >= 80) {
    std::cout << "B";
} else {
    std::cout << "C or below";
}
```

### For Loop
```cpp
for (int i = 0; i < 5; ++i) {
    std::cout << i << " ";
}
// Output: 0 1 2 3 4
```

### While Loop
```cpp
int n = 3;
while (n > 0) {
    std::cout << n << std::endl;
    --n;
}
```

## Functions
Break your code into reusable pieces.

```cpp
int add(int a, int b) {
    return a + b;
}

int main() {
    std::cout << add(2, 3); // 5
}
```

## 1.4 Control Flow

### If-Else
```cpp
if (score >= 90) {
    std::cout << "A";
} else if (score >= 80) {
    std::cout << "B";
} else {
    std::cout << "C or below";
}
```

### Loops
```cpp
// For loop
for (int i = 0; i < 10; i++) {
    std::cout << i << " ";
}

// While loop
int n = 5;
while (n > 0) {
    std::cout << n-- << " ";
}

// Range-based for (C++11+)
std::vector<int> nums = {1, 2, 3, 4, 5};
for (const auto& num : nums) {
    std::cout << num << " ";
}
```

## 1.5 Functions & Lambdas

Functions group reusable logic. Lambdas are anonymous functions (C++11+).

```cpp
int add(int a, int b) {
    return a + b;
}

// Lambda
auto multiply = [](int a, int b) { return a * b; };
std::cout << multiply(3, 4); // 12
```

---

## 1.6 Scope, Lifetime, and Const

Variables declared inside `{}` are local to that block. Use `const` for values that shouldn’t change.

```cpp
const double PI = 3.14159;
```

---

## 1.7 Error Handling

Use exceptions for error handling.

```cpp
#include <stdexcept>
int divide(int a, int b) {
    if (b == 0) throw std::runtime_error("Divide by zero");
    return a / b;
}
```

## 1.8 Best Practices & Pitfalls

- Always initialize variables
- Prefer `std::string` over C-style strings
- Use `auto` for iterators and complex types
- Avoid using `using namespace std;` in headers
- Watch for integer division (e.g., `5/2 == 2`)

## 1.12 Advanced: Namespaces & Header Files

Namespaces help organize code and avoid name conflicts. Header files (`.h`/`.hpp`) let you split code into reusable modules.

```cpp
// math_utils.h
namespace Math {
    int add(int a, int b);
}

// math_utils.cpp
#include "math_utils.h"
int Math::add(int a, int b) { return a + b; }

// main.cpp
#include "math_utils.h"
int main() {
    std::cout << Math::add(2, 3);
}
```

## Prerequisites

<Warning>
**Before starting this course**, you should be comfortable with:

- Basic terminal/command line usage
- Git version control basics

## Development Environment

<Tabs>
  <Tab title="Required">
    - **Node.js 18+** - JavaScript runtime
    - **npm or yarn** - Package manager
    - **VS Code** - Recommended editor
    - **Expo Go app** - For testing on device
  </Tab>
  <Tab title="iOS Development">
    - **macOS** - Required for iOS development
    - **Xcode 14+** - iOS simulator and build tools
    - **Apple Developer Account** - For App Store deployment
  </Tab>
  <Tab title="Android Development">
    - **Android Studio** - Android SDK and emulator
    - **Java 11+** - Required for Android builds
    - **Google Play Console** - For Play Store deployment
  </Tab>
</Tabs>

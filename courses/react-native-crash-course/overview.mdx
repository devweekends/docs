    **Focus:** Fundamentals and core concepts
    
    1. Complete Modules 1-8 (Getting Started through Networking)
    2. Build 2-3 simple apps to practice
    3. Understand React Native lifecycle
    4. Master core components and styling
    
    **Time Commitment:** 10-15 hours/week
  </Tab>
  
  <Tab title="Intermediate (2-4 months)">
    **Focus:** Advanced features and native APIs
    
    1. Complete Modules 9-16 (Storage through Performance)
    2. Build apps using native features
    3. Learn state management libraries
    4. Optimize app performance
    
    **Time Commitment:** 10-15 hours/week
  </Tab>
  
  <Tab title="Advanced (4-6 months)">
    **Focus:** Production-ready apps and deployment
    
    1. Complete Modules 17-26 (Testing through Capstone)
    2. Deploy apps to stores
    3. Learn architecture patterns
    4. Build complete capstone project
    
    **Time Commitment:** 15-20 hours/week

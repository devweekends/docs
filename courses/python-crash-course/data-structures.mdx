# Data Structures

## Lists
```python
numbers = [1, 2, 3, 4, 5]

# Operations
numbers.append(6)
numbers.insert(0, 0)
numbers.remove(3)
numbers.pop()

# Slicing
numbers[1:4]  # [2, 3, 4]
numbers[:3]   # [1, 2, 3]
numbers[-1]   # Last element
```

## List Comprehensions
```python
# Basic
squares = [x**2 for x in range(10)]

# With condition
evens = [x for x in range(10) if x % 2 == 0]

# Nested
matrix = [[i*j for j in range(3)] for i in range(3)]
```

## Dictionaries
```python
person = {
    "name": "John",
    "age": 25,
    "city": "New York"
}

# Accessing
name = person["name"]
age = person.get("age", 0)

# Dictionary comprehension
squares = {x: x**2 for x in range(5)}
```

## Sets
```python
numbers = {1, 2, 3, 4, 5}

# Operations
numbers.add(6)
numbers.remove(3)

# Set operations
a = {1, 2, 3}
b = {3, 4, 5}

a | b  # Union
a & b  # Intersection
a - b  # Difference
```

## Collections Module
```python
from collections import Counter, defaultdict, deque

# Counter
counter = Counter(["apple", "banana", "apple"])
counter.most_common(1)  # [('apple', 2)]

# defaultdict
dd = defaultdict(list)
dd['key'].append(1)

# deque
dq = deque([1, 2, 3])
dq.appendleft(0)
dq.append(4)
```

# Python Fundamentals

## Hello World
```python
print("Hello, World!")
```

## Data Types
```python
age = 25
price = 19.99
name = "John"
is_active = True
value = None
```

## Control Flow
```python
# If-else
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")

# For loop
for i in range(10):
    print(i)

# For with enumerate
for index, name in enumerate(names):
    print(f"{index}: {name}")

# While loop
while condition:
    pass
```

## Functions
```python
def greet(name):
    return f"Hello, {name}!"

# Default parameters
def greet(name="Guest"):
    return f"Hello, {name}!"

# *args and **kwargs
def sum_all(*args):
    return sum(args)

# Lambda
add = lambda a, b: a + b

# Type hints
def add(a: int, b: int) -> int:
    return a + b
```

## String Formatting
```python
name = "John"
age = 25

# f-strings
message = f"My name is {name} and I'm {age} years old"

# format method
message = "My name is {} and I'm {} years old".format(name, age)
```

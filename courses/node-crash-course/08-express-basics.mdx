# Express.js Basics

While the built-in `http` module is powerful, it requires a lot of boilerplate code for complex applications. **Express.js** is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

## Installation

```bash
npm install express
```

## Creating a Basic Server

```javascript
const express = require('express');
const app = express();
const PORT = 5000;

app.get('/', (req, res) => {
  res.send('Hello from Express!');
});

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});
```

## Routing

Express makes routing much easier than the raw `http` module.

```javascript
// GET request
app.get('/about', (req, res) => {
  res.send('About Page');
});

// POST request
app.post('/contact', (req, res) => {
  res.send('Contact Form Submitted');
});

// Dynamic Parameters
app.get('/users/:id', (req, res) => {
  res.send(`User ID: ${req.params.id}`);
});
```

## Middleware

Middleware functions are functions that have access to the request object (`req`), the response object (`res`), and the next middleware function in the applicationâ€™s request-response cycle (`next`).

### Creating Custom Middleware

```javascript
const logger = (req, res, next) => {
  console.log(`${req.method} ${req.protocol}://${req.get('host')}${req.originalUrl}`);
  next(); // Call next middleware
};

// Initialize middleware
app.use(logger);
```

### Built-in Middleware

Express has built-in middleware for parsing body data.

```javascript
// Parse JSON bodies
app.use(express.json());

// Parse URL-encoded bodies (forms)
app.use(express.urlencoded({ extended: false }));
```

## Serving Static Files

To serve static files such as images, CSS files, and JavaScript files, use the `express.static` built-in middleware function.

```javascript
const path = require('path');

// Set 'public' folder as static folder
app.use(express.static(path.join(__dirname, 'public')));
```

## Summary

- **Express.js** simplifies server creation and routing.
- Use `app.get()`, `app.post()`, etc., for routing.
- **Middleware** allows you to execute code between the request and response.
- `express.json()` is essential for parsing JSON data from requests.
- `express.static()` serves static assets easily.

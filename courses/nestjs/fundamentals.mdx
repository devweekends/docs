# NestJS Fundamentals

## Installation
```bash
npm install -g @nestjs/cli
nest new my-app
```

## Controllers
```typescript
@Controller('users')
export class UsersController {
  @Get()
  findAll() {
    return 'All users';
  }
}
```

## Services
```typescript
@Injectable()
export class UsersService {
  findAll() {
    return [];
  }
}
```

## Modules
```typescript
@Module({
  controllers: [UsersController],
  providers: [UsersService],
})
export class UsersModule {}
```

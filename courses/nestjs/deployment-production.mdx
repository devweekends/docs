# Deployment & Production

## Docker
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build
CMD ["node", "dist/main"]
```

## Health Checks
```typescript
@Get('health')
@HealthCheck()
check() {
  return this.health.check([
    () => this.db.pingCheck('database'),
  ]);
}
```

## Logging
```typescript
private readonly logger = new Logger(AppService.name);

doSomething() {
  this.logger.log('Doing something...');
}
```

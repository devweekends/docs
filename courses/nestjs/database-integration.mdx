# Database Integration

## TypeORM Setup
```typescript
TypeOrmModule.forRoot({
  type: 'postgres',
  host: 'localhost',
  port: 5432,
  database: 'mydb',
  entities: [User],
  synchronize: true,
})
```

## Entities
```typescript
@Entity()
export class User {
  @PrimaryGeneratedColumn()
  id: number;
  
  @Column()
  name: string;
  
  @OneToMany(() => Post, post => post.author)
  posts: Post[];
}
```

## Transactions
```typescript
await this.dataSource.transaction(async manager => {
  await manager.save(user);
  await manager.save(profile);
});
```

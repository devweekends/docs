## What is AWS?

Amazon Web Services (AWS) is the world's most comprehensive cloud platform, offering over 200 services from data centers globally. It powers everything from startups to Fortune 500 companies.

<CardGroup cols={3}>
  <Card title="Compute" icon="server">

  </Card>
  <Card title="Storage" icon="database">
    S3, EBS, EFS for data storage solutions
  </Card>
  <Card title="Networking" icon="network-wired">
    VPC, Route 53, CloudFront for connectivity
  </Card>
</CardGroup>

<Note>
**AWS Dominance**: AWS holds ~32% of the cloud market share, making it the most in-demand cloud skill for engineers.
</Note>

## üîß Key Services Overview

### Compute
| Service | Purpose | Use Case |
|---------|---------|----------|
| EC2 | Virtual servers | Web servers, apps |
| Lambda | Serverless functions | Event-driven processing |
| ECS/EKS | Container orchestration | Microservices |
| Elastic Beanstalk | PaaS | Quick deployments |

### Storage
| Service | Purpose | Use Case |
|---------|---------|----------|
| S3 | Object storage | Files, backups, static sites |
| EBS | Block storage | EC2 volumes |
| EFS | File storage | Shared file systems |
| Glacier | Archive storage | Long-term backups |

### Database
| Service | Type | Use Case |
|---------|------|----------|
| RDS | Relational | MySQL, PostgreSQL, SQL Server |
| DynamoDB | NoSQL | High-scale key-value |
| ElastiCache | In-memory | Redis, Memcached caching |
| Aurora | Relational | High-performance MySQL/PostgreSQL |

### Networking
| Service | Purpose | Use Case |
|---------|---------|----------|
| VPC | Virtual network | Isolated cloud network |
| Route 53 | DNS | Domain management |
| CloudFront | CDN | Content delivery |
| ELB | Load balancing | Traffic distribution |

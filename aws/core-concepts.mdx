**Key Points**:

- Failure in one AZ doesn't affect others

### Edge Locations

**Edge Locations** are AWS data centers designed to deliver content to end users with low latency. Used by:
- **CloudFront** (CDN)
- **Route 53** (DNS)
- **AWS Global Accelerator**

There are 400+ edge locations worldwide (more than regions).

## AWS Pricing Models

### On-Demand
Pay for compute capacity by the hour/second with no commitments.

```python
# Example: EC2 On-Demand pricing
t3_medium_hourly = 0.0416  # USD per hour
monthly_cost = t3_medium_hourly * 24 * 30  # ~$30/month
```

**Best for**: Unpredictable workloads, development, short-term projects

### Reserved Instances (RI)
Commit to 1 or 3 years for significant discounts (up to 72% off).

| Term | Payment | Discount |
|------|---------|----------|
| 1 Year | No Upfront | ~36% |
| 1 Year | All Upfront | ~42% |
| 3 Year | All Upfront | ~72% |

**Best for**: Steady-state workloads, production databases

### Spot Instances
Bid on unused EC2 capacity for up to 90% discount.

```python
# Spot pricing example
on_demand_price = 0.10  # USD/hour
spot_price = 0.03       # Up to 70% cheaper!

# But instances can be terminated with 2-min warning
```

**Best for**: Batch processing, fault-tolerant workloads, CI/CD

### Savings Plans
Flexible pricing with commitment to consistent usage ($/hour).

**Best for**: Consistent compute usage across EC2, Lambda, Fargate
